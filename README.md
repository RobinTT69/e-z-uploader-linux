# E-Z Uploader


Make sure your screenshot app (I use flameshot) saves its screenshots in ~/screenshots/


## Reqs

- pnpm
- node v20.10.0
- rust 1.57

## Dev

```
pnpm install
pnpm tauri dev
```

## Build

```
pnpm install
pnpm tauri build
```
